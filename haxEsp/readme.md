# haxl

Haxl es una biblioteca de Haskell que simplifica el acceso a datos remotos, como bases de datos o servicios basados en la web. Haxl puede procesar automáticamente varias solicitudes en la misma fuente de datos, solicitar datos de múltiples fuentes de datos simultáneamente y almacenar solicitudes anteriores.

Tener todo esto manejado para usted detrás de las escenas significa que su código de búsqueda de datos puede ser mucho más limpio y más claro de lo que sería de otro modo si tuviera que preocuparse por la optimización de la búsqueda de datos.
